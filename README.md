## Dark.nvim

:moon: A dark colorscheme for [nvim](https://neovim.io/)

### Install

As usual using your plugin manager, e.g. lazy.nvim

```lua
local P = {
  'doums/dark.nvim',
  lazy = true,
  priority = 1000,
}

return P
```

Then load the colorscheme in your config

```lua
vim.cmd('colorscheme dark')
```
